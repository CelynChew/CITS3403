Welcome to the Tutorial Page

Login or Register An Account



If you do not have one already, Chatroom requires it's users to register an account.

Select a unique username and a unique password that only you know to avoid any malicious attacks!

Create a Chat



Once logged in, click the plus button then enter the username of who you'd like to chat with to create a chat.

Enjoy!



Begin sending messages! Chatroom will save all chats created allowing you to use whenever you feel necessary!

Use the trash button to delete chats and you can also upload files using the file upload button!

Make sure to logout when you are finished!
